# Molecule
RSS / Atom Feed Blog Aggregator

## Dependencies

- NodeJS

## Installation

```sh
sudo npm install -g molecule-js
```

## Usage

```sh
$ molecule --help

molecule [options] <config directory>

Options:
  --pretty, -p
    Disables minification
  --output, -o <build directory>
    Changes ./build location.
    Default: config directory
```
